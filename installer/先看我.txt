TaskBeacon 安装说明
========================================

这是什么：一个 macOS 菜单栏小工具，把你所有正在跑的 Claude Code 终端会话
集中盯住——一眼看清哪个「需要你确认 / 正在跑 / 已完成」，点一下跳回对应终端。

需要满足：
  • macOS 13 或更高
  • 你是 Claude Code 用户（命令行 `claude`）
  • 用 VSCode 跑 Claude（精准跳转终端依赖 VSCode 扩展；不用 VSCode 也能看状态）


安装（两步）
----------------------------------------
1. 双击本文件夹里的  install.command

2. 第一次双击若被 macOS 拦截（提示「无法验证开发者」）：
   右键点击 install.command → 选「打开」 → 弹窗里再点一次「打开」。

   安装器会自动：装好 app、装好 hook、接进你的 Claude 配置、装好 VSCode 扩展。
   完成后菜单栏右上角会出现 TaskBeacon 图标。

3. 让「精准跳转终端」生效：回到 VSCode，按 Cmd+Shift+P → 输入并执行
   Developer: Reload Window（只需一次）。


卸载
----------------------------------------
双击  uninstall.command  即可干净移除（app + hook + 配置 + 扩展）。


说明
----------------------------------------
• 这个 app 没有花钱做 Apple 公证，所以首次打开需要上面的右键「打开」步骤，
  属正常现象，不是病毒。安装器源码（install.command）是纯文本，可自行查看。
• 安装器只往 ~/.claude 和 ~/.vscode/extensions 写文件，不碰系统其它地方。
